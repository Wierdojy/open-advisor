# Open Advisor Web App

Runnable MVP web UI using Node's built-in HTTP server and static assets.

Implemented surfaces:
- Home digest
- Portfolio
- Theses
- Calendar
- Research

Implemented flows:
- add holdings
- add watchlists
- create theses
- add catalysts
- create research sweeps
- mark alerts seen
- snooze alerts
- reset demo data

Purpose of this slice:
- prove the thesis-driven market copilot loop
- stay faithful to the architecture before heavier framework choices
- keep the seam clean for later migration to React/Next or a richer frontend stack
